# 🧠 SQL Server Expert Portfolio

---

## 📌 Sobre o Projeto

Este repositório documenta minha jornada prática de aprendizado em SQL Server, com foco em atuação como:

- Analista de Dados  
- DBA Júnior  

O conteúdo é baseado na trilha SQL Server On-Premise, abordando desde fundamentos de infraestrutura até tópicos avançados como alta disponibilidade e replicação.

---

## 🎯 Objetivo

Desenvolver conhecimento técnico sólido em SQL Server, indo além de queries, com foco em:

- Infraestrutura de banco de dados  
- Administração e manutenção  
- Segurança  
- Performance  
- Monitoramento
